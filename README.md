- # loseq-audio-memo
	- 一个能够在 logseq 中录音并插入文档中的插件
	- ![image.png](./assets/home.jpg)
- ## 功能
	- 录音
	- 播放
	- 下载
	- 插入至文档
- ## 待办

	- [ ] 支持其他格式的音频文件（比如： mp3）
	- [ ] 双语界面（目前仅支持中文）
